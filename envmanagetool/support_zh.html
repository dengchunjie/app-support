应用简介

Env Manage Tool 是一款专业的服务器环境配置管理工具，支持 iOS 和 macOS 双平台。它帮助开发者、运维工程师和系统管理员安全地存储和管理服务器 SSH 凭据、Web 管理后台登录信息以及 SSL 证书文件。

所有敏感数据均使用 AES-256-GCM 加密，并通过 Face ID / Touch ID 保护访问。您的数据通过 iCloud 在所有 Apple 设备间无缝同步。

核心功能

encrypted

AES-256-GCM 加密

所有密码和敏感凭据在存储前均在本地加密处理。

fingerprint

生物识别认证

支持 Face ID 或 Touch ID 解锁，可配置自动锁定时间以增强安全性。

cloud_sync

iCloud 同步

环境配置和设置通过 CloudKit 在所有 Apple 设备间自动同步。

folder_managed

按客户分组管理

按客户或项目分组管理环境配置，支持生产、测试、开发三种环境类型。

content_copy

一键复制

轻点即可复制主机地址、用户名或密码。macOS 上还支持顺序复制模式，工作流程更高效。

import_export

导入与导出

支持将环境配置导出为 JSON 文件进行备份或分享，也可从 JSON 文件导入配置。

常见问题

我的数据如何被保护？

所有敏感数据（密码、Web 凭据）在存储前均使用 AES-256-GCM 加密。应用通过生物识别认证（Face ID / Touch ID）或主密码保护访问。应用未集成任何第三方 SDK 或分析工具。

iCloud 同步如何工作？

您的环境数据通过 Apple CloudKit 自动同步。所有数据存储在您的私有 iCloud 容器中，开发者无法访问您的数据。只要有网络连接，同步就会在后台自动进行。

什么是顺序复制模式（macOS）？

顺序复制是 macOS 独有的功能，通过键盘快捷键自动按顺序复制主机地址、用户名和密码。使用此功能需要在系统设置中授予辅助功能权限。

iOS 版本是免费的吗？

是的，iOS 版本完全免费，所有功能均已解锁。macOS 版本需要年度订阅，提供 7 天免费试用。

可以导出我的数据吗？

可以。您可以将全部或选定的环境配置导出为 JSON 文件，用于备份或分享。您也可以将 JSON 配置文件导入到应用中。

可以存储哪些类型的信息？

每个环境条目支持：服务器 SSH 详情（主机、端口、用户名、密码、路径）、Web 管理后台凭据（访问地址、用户名、密码）、SSL 证书文件附件、环境类型（生产 / 测试 / 开发）以及备注。
