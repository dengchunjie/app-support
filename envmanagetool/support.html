App Overview

Env Manage Tool is a professional server environment configuration management app for iOS and macOS. It helps developers, DevOps engineers and system administrators securely store and manage server SSH credentials, web admin panel logins and SSL certificate files.

All sensitive data is encrypted with AES-256-GCM and protected by Face ID / Touch ID. Your data syncs seamlessly across all your Apple devices via iCloud.

Frequently Asked Questions

How is my data secured?

All sensitive data (passwords, web credentials) is encrypted using AES-256-GCM before being stored. The app uses biometric authentication (Face ID / Touch ID) or a master password to protect access. No third-party SDKs or analytics tools are integrated.

How does iCloud sync work?

Your environment data syncs automatically via Apple CloudKit. All data is stored in your private iCloud container - the developer cannot access your data. Sync happens in the background whenever a network connection is available.

What is Sequential Copy mode (macOS)?

Sequential Copy is a macOS-exclusive feature that automatically copies host, username, and password in sequence using keyboard shortcuts. This requires granting Accessibility permission in System Settings.

Is the iOS version free?

Yes, the iOS version is completely free with all features unlocked. The macOS version requires an annual subscription with a 7-day free trial.

Can I export my data?

Yes. You can export all or selected environments as JSON files for backup or sharing. You can also import JSON configurations into the app.

What types of information can I store?

Each environment entry supports: server SSH details (host, port, username, password, path), web admin panel credentials (URL, username, password), SSL certificate file attachments, environment type (Prod / Test / Dev), and notes.
